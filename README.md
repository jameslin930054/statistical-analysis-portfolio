# Statistical Analysis & Machine Learning Portfolio

本倉庫整理多個與統計分析、機器學習、深度學習及大型語言模型（LLM）相關的實作專案。

內容涵蓋傳統機器學習模型、降維方法、深度神經網路、電腦視覺應用，以及基於 Hugging Face Transformer 的大型語言模型分析與實作。

## Project Categories

### Statistical Analysis & Machine Learning

包含統計分析方法與傳統機器學習模型實作：

- Logistic Regression
- KNN / LDA / QDA 分類模型
- PCA / SVD 降維分析
- 分類器效能比較與實驗分析

相關專案：
- LogisticRegression/
- Machine_Learning_Practice/
- PCA/

- 分類器的評比實驗/

### Deep Learning & Computer Vision

包含深度學習模型建立與影像相關應用：

- PyTorch 深度學習模型實作
- CNN 影像分類
- SRCNN 超解析度重建模型

相關專案：
- PyTorch/
- SRCNN/
- 卷積神經網路 CNN 在影像辨識與超解析生成/


---

### Large Language Model (LLM) & Hugging Face

本專案透過 Hugging Face Transformers 實作大型語言模型分析流程，探索 Transformer 架構內部運作機制。

主要內容包含：

- 使用 Hugging Face Hub 載入預訓練語言模型
- Tokenizer 與 Model 初始化流程
- Token Embedding 分析
- 使用 Cosine Similarity 比較 Token 之間的語意關聯
- Transformer Hidden Representation 分析
- 使用 Logit Lens 觀察不同 Layer 的模型預測傾向

技術使用：

- Python
- PyTorch
- Hugging Face Transformers
- Hugging Face Hub
- Transformer Architecture
- Llama-based Language Model

相關專案：
LLM/
